## INSTALLATION INSTRUCTIONS 

```js
 $ git clone https://github.com/nathvarun/Uber-App-UI-Clone-React-Native.git
 $ cd /Uber-App-UI-Clone-React-Native
 $ npm install 
```

## iOS 
```js
 $ react-native run-ios
```

## Android
```js
 $ react-native run-android
```
